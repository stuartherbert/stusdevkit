# DependencyKit for Stu's Dev Kit

A PSR-compatible dependency injection solution for PHP 8.5+.

Part of Stu's Dev Kit: building blocks for assembling the things you need to build, in a way that will last.

## Installation

```bash
composer require stusdevkit/dependencykit
```

## Scope

DependencyKit provides a dependency injection container that conforms to [PSR-11 Container Interface](https://www.php-fig.org/psr/psr-11/), plus supporting utilities for wiring together application dependencies.
